# flex 实现表格布局

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Flex 实现表格</title>
    <style>
      /* 1. 外部表格容器设置固定宽度 */
      .flex-table {
        width: 100%;
        max-width: 800px;
        margin: 20px auto;
        border: 1px solid #ddd;
        border-radius: 4px;
        overflow: hidden;
        /* 隐藏边框圆角外的内容 */
      }

      /* 2. 表格行 display: flex; */
      .table-row {
        display: flex;
        width: 100%;
        border-bottom: 1px solid #ddd;
      }

      /* 3. 表格设置 flex: 1; 均分宽度 */
      .table-cell {
        padding: 12px;
        flex: 1;
        /* 均分宽度（也可设置固定宽度，如 width: 200px） */
        text-align: center;
        border-right: 1px solid #ddd;
        box-sizing: border-box;
        /* 确保padding不影响宽度 */
      }

      /* 最后一行去掉下边框 */
      .table-row:last-child {
        border-bottom: none;
      }

      /* 表头行样式 */
      .table-header {
        background-color: #f5f5f5;
        font-weight: bold;
      }

      /* 最后一列去掉右边框 */
      .table-cell:last-child {
        border-right: none;
      }

      /* 隔行变色（可选） */
      .table-row:nth-child(even) {
        background-color: #fafafa;
      }

      /* 响应式适配（可选）- 小屏幕下单元格换行 */
      @media (max-width: 600px) {
        .table-row {
          flex-direction: column;
        }

        .table-cell {
          border-right: none;
          border-bottom: 1px solid #ddd;
          text-align: left;
        }

        .table-cell:last-child {
          border-bottom: none;
        }
      }
    </style>
  </head>

  <body>
    <div class="flex-table">
      <!-- 表头行 -->
      <div class="table-row table-header">
        <div class="table-cell">ID</div>
        <div class="table-cell">姓名</div>
        <div class="table-cell">年龄</div>
        <div class="table-cell">城市</div>
      </div>
      <!-- 内容行1 -->
      <div class="table-row">
        <div class="table-cell">1</div>
        <div class="table-cell">张三</div>
        <div class="table-cell">25</div>
        <div class="table-cell">北京</div>
      </div>
      <!-- 内容行2 -->
      <div class="table-row">
        <div class="table-cell">2</div>
        <div class="table-cell">李四</div>
        <div class="table-cell">30</div>
        <div class="table-cell">上海</div>
      </div>
      <!-- 内容行3 -->
      <div class="table-row">
        <div class="table-cell">3</div>
        <div class="table-cell">王五</div>
        <div class="table-cell">28</div>
        <div class="table-cell">广州</div>
      </div>
    </div>
  </body>
</html>

# CSS 矩形旋转

transform: rotate(45deg);

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>矩形旋转示例</title>
    <style>
      .container {
        display: flex;
        justify-content: center;
        align-items: center;
        height: 80vh;
      }

      .rectangle {
        width: 200px;
        height: 100px;
        background-color: #3498db;
        border: 2px solid #2980b9;

        /* 旋转45度 */
        transform: rotate(45deg);

        /* 可选：添加过渡效果 */
        transition: transform 5s ease;

        /* 确保旋转中心在元素中心 */
        transform-origin: center;

        display: flex;
        justify-content: center;
        align-items: center;
        color: white;
        font-weight: bold;
      }

      /* 悬停时旋转到不同角度 */
      .rectangle:hover {
        transform: rotate(135deg);
      }
    </style>
  </head>
  <body>
    <div class="container">
      <div class="rectangle">旋转矩形</div>
    </div>
  </body>
</html>

# 三栏 Flexbox 方案

<div class="container">
  <div class="left">Left (200px)</div>
  <div class="center">Center (自适应)</div>
  <div class="right">Right (200px)</div>
</div>

.container {
  display: flex;
  height: 100vh;
}

.left,
.right {
  flex: 0 0 200px; /* 不放大，不缩小，固定200px */
  background: #f0f0f0;
}

.center {
  flex: 1; /* 占据剩余空间 */
  background: #e0e0e0;
}

# 三栏 grid 方案

<div class="container">
  <div class="left">Left (200px)</div>
  <div class="center">Center (自适应)</div>
  <div class="right">Right (200px)</div>
</div>

.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px; /* 固定 - 自适应 - 固定 */
  height: 100vh;
}

.left,
.right {
  background: #f0f0f0;
}

.center {
  background: #e0e0e0;
}

# 水平垂直居中 Flexbox

.container {
  display: flex;

  align-items: center; /* 垂直居中 */
  height: 100vh; /* 需要明确高度 */
}

# relative、absolute、fix

relative 用于相对当前文档流定位。
absolute 用于从文档流中移除，并相对于最近的定位祖先定位。
fixed 用于从文档流中移除，并相对于视窗固定定位，无论页面如何滚动都不会改变。

# 元素居中

/* 用于内联级元素和文本。 */
.container {
  text-align: center;
}

/* 用于块级元素和弹性布局 */
.container {
  display: flex;
  justify-content: center;
}

/* 用于块级元素和网格布局 */
.container {
  display: grid;
  justify-items: center;
}

/* 用于绝对定位的元素 */
.container {
  position: relative;
}

.centered {
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}

# @Media 用途

  • 响应式设计：通过 @media，开发者可以为不同设备、屏幕尺寸或条件创建特定的样式，从而使网页在多种环境中表现良好。

  • 多样性：@media 支持多种条件，如设备类型、屏幕宽度、高度、分辨率、颜色深度等。这使得样式定制非常灵活。

# 常见条件

  • min-width 和 max-width：用于根据屏幕或视窗的宽度应用样式。常用于响应式设计中，确保页面在不同屏幕尺寸上都能正确显示。

  • min-height 和 max-height：根据高度进行样式调整。

  • orientation：根据设备的方向（横向或纵向）应用样式。

    • landscape 横向
    • protrait 纵向

  • print：为打印时提供定制样式。

  • screen：适用于显示屏幕的样式

# Demo

/* 为宽度大于等于 600px 的屏幕应用特定样式 */
@media (min-width: 600px) {
  .container {
    width: 50%;
  }
}

/* 为打印应用特定样式 */
@media print {
  .no-print {
    display: none;
  }
}

/* 在这个例子中，只有当宽度大于600px且设备为横向时，.landscape-container 才会显示。 */
@media (min-width: 600px) and (orientation: landscape) {
  .landscape-container {
    display: block;
  }
}
